# Just a basic python script to set the current level to whatever you want 
Stuck at level 3? tried everything?? Here you go...

## Requirements:
**Python:**
Install via the Windows terminal
```
winget install python3
```
Or direct download from: 
[Here](https://www.python.org/ftp/python/3.13.9/python-3.13.9-amd64.exe)

## Instructions
1 - Download the .py file in this repo.
2 - Copy the file to: 
```
C:\Users\%username%\AppData\Local\BET\Saved\SaveGames
```
3 - In the same directory right click and "Open in terminal" this is important because if you run it with python it'll launch from a different directory and **will not** work, unless you provide the full path.
4 - run the following command:
```
python .\level_setter.py
```
5 - Type or copy the name and extension of your savagame, it'll be the .sav file (e.g BOOSTED.sav)

6 - Type the desired filename, if left empty it'll be "originalfilename_patched.sav"

7 - enter the lore the level that you want, available levels [Here](https://backrooms-escape-together.fandom.com/wiki/Levels)
  
8 - GG Have fun!
